Для запуска кода необходимо запустить программу cloud.py. Программа считывает статьи из базы данных и выбирает три рекомендованных по одной выбранной пользователем.

Статьи добавляются в базу как обычные текстовые файлы, со скопированными ссылкой, названием и текстом статьи, в кодировке 1251(ANSI).

В dataframe выгружаются все ключевые слова, по которым выбираются статьи.
